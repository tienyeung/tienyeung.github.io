斐波那契数列的优化

2019-10-18 Views 编程之美 5 min read             繁
 

斐波那契数列是一个非常美丽、和谐的数列，有人说它起源于一对繁殖力惊人、基因非常优秀的兔子，也有人说远古时期的鹦鹉螺就知道这个规律。

常规递归法

def Fibonacci(n):
    if n<=0:
        return 0
    elif n == 1:
        return 1
    else:
        return  Fibonacci(n-1)+Fibonacci(n-2)

思考一下，能否将其效率进一步优化，递归法其中涉及到许多重复计算，而如何减少这些计算呢？可以用数组储存所有已计算过的项，这样便可以达到空间换时间的目的。但是否有更快的算法呢？

常规法备忘录

def fib(self, N):
        """
        :type N: int
        :rtype: int
        """
        if N<=0:
            return 0
        elif N == 1 or N == 2:
            return 1
        f1,f2,fn=1,1,0
        for i in range(3,N+1):
            fn = f1 + f2
            f1 = f2
            f2 = fn
        return fn

动态规划

int fib(int N) {
    vector<int> dp(N + 1, 0);
    dp[1] = dp[2] = 1;
    for (int i = 3; i <= N; i++)
        dp[i] = dp[i - 1] + dp[i - 2];
    return dp[N];
}

func fib(N int) int {
	if N == 0 || N == 1 {
		return N
	}
	// first 是第0个和第1个元素
	first := [][2]int{[2]int{0, 1}, [2]int{0, 0}}
	// temp为系数
	temp := [][2]int{[2]int{0, 1}, [2]int{1, 1}}
	res := matrix22_pow(temp, N - 1)
	return matrix22_mul(first, res)[0][1]
}

func matrix22_pow(x [][2]int, n int) [][2]int {
	r := x
	res := [][2]int{[2]int{1, 0}, [2]int{0, 1}}
	for n != 0 {
		if n & 1 == 1 {
			// 最低二进制位为1
			res = matrix22_mul(res, r)
		}
		// 2维矩阵相乘
		r = matrix22_mul(r, r)
		n >>= 1
	}
	return res
}

func matrix22_mul(x, y [][2]int) [][2]int {
	temp := make([][2]int, 2)
	temp[0][0] = x[0][0] * y[0][0] + x[0][1] * y[0][1]
	temp[0][1] = x[0][0] * y[0][1] + x[0][1] * y[1][1]
	temp[1][0] = x[1][0] * y[0][0] + x[1][1] * y[0][1]
	temp[1][1] = x[1][0] * y[1][0] + x[1][1] * y[1][1]
	return temp
}
